Zero system dependencies — Chromium and ffmpeg bundled via npm install. Works on macOS, Linux, Windows. Node 20+.

How it works

Frame-accurate, not screen-capture

1

Trap React

Intercept window.React before Claude Design's UMD bundle runs. Sniff every createElement for the timeline context shape.

2

Drive the timeline

Once we have setTime, Playwright steps the animation frame by frame at 2× supersampling.

3

Encode

Pipe PNGs to ffmpeg. Lanczos downscale, H.264 CRF 15, BT.709, +faststart. Bit-identical regardless of machine speed.
